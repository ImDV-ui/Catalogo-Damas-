
# Catálogo Web — Damas Candy Store (plantilla)

Este proyecto cumple los requisitos mínimos: **HTML, CSS, JSON y JavaScript (fetch)**.
Incluye una cuadrícula de productos con **búsqueda, filtros por categoría y ordenación**, y diseño **responsive** (móvil/tablet/desktop).

## Estructura
```
damas-catalog/
├── index.html
├── styles.css
├── app.js
├── data/
│   └── products.json
└── assets/
    ├── bg.jpg                # Fondo proporcionado
    └── logo-placeholder.svg  # Sustituir por vuestro logo (PNG/SVG)
```

> Opcional: coloca tu `logo.png` en `assets/` y cambia la ruta en `index.html`.

## Cómo usar
1. Abre `index.html` en el navegador (doble clic).
2. Para editar el catálogo, modifica `data/products.json` (nombre, precio, imagen y categoría).
3. Las imágenes ahora usan placeholders `https://picsum.photos/...`. Reemplázalas por URLs reales o por archivos locales en `assets/` y apunta la ruta en el JSON.

## Colores y estilo
Se ha usado el fondo lime con espiral y acentos morados para acercarse al estilo de Instagram/branding enviado.
